Widget Preview

Test the Treasure Coast AI chat widget on this page. Look for the chat bubble in the bottom-right corner.

How to Embed on Your Website

Copy and paste this code just before the closing </body> tag:

<script src="https://YOUR_DOMAIN/widget/embed.js" data-client-id="YOUR_CLIENT_ID" data-bot-id="YOUR_BOT_ID" data-primary-color="#2563eb" data-position="bottom-right" ></script>

Configuration Options

  • data-client-id - Your unique client identifier
  • data-bot-id - The bot you want to display
  • data-primary-color - Brand color (hex code)
  • data-position - Widget position (bottom-right, bottom-left, top-right, top-left)
  • data-greeting - Custom welcome message
Note: This preview uses the Faith House demo bot. The chat bubble should appear in the corner. Click it to open the chat window.
